Raspberry Pi Project

It all started with some silicon

I recieved a Raspberry Pi from a friend and wanted to make it into a retro gaming station.

The software portion of this project was fairly straight forward. I used a pre-made operating system image from RetroPie, and I had to do just a little bit of fiddling with the kernel modules to get the RPi to talk to the controller I intended to use.

I then had to create my own connectors for the controller.

And wire everything up

Great Success!
